# 20210125 BARE REPO

bare 是管理仓库的直接方式.

与一般仓库不同的是, bare 不会包含任何 work tree 里的内容,

只会展开 work tree 中的.git 文件夹内容,

从而通过 git 其他操作对历史内容进行修正

```bash
git clone https://xx.com --bare
```

.git 内容是自 git init 始 存在于仓库之中

记录了仓库 history 和版本信息

```bash
git init --bare
```
